# User Flow

## New User Flow

```text
Register
  v
Choose Connect Demo Bank or Manual Setup
  v
Set current balance
  v
Set payday and income
  v
Add protected essentials
  v
Dashboard calculates safe-to-spend and Money Pressure
  v
Mini Guard preview shows quick status
```

## Returning User Flow

```text
Login
  v
Dashboard
  v
See Safe to Spend / Recovery Gap
  v
Check Money Pressure, confidence and last synced time
  v
Update balance / sync demo bank / import CSV if needed
  v
Dashboard recalculates
```

## Demo Bank Flow

```text
Dashboard
  v
Connect Bank
  v
Choose Freedom, Watch, Recovery, Student Worker or Renter demo
  v
Sync demo account
  v
Dashboard updates Safe to Spend, Money Mode, Money Pressure, Transactions and Mini Guard
```

## Spending Decision Flow

```text
User wants to spend
  v
Checks Mini Guard or Dashboard
  v
If safe amount is enough:
    spend confidently
  v
If safe amount is low:
    Watch Mode warning
  v
If safe-to-spend is negative:
    Recovery Mode action
```

## Recovery Flow

```text
Recovery Mode appears
  v
User sees Recovery Gap
  v
Short Recovery Plan protects food, transport and rent first
  v
User pauses non-essential spending until payday
```

## Settings / Account Flow

```text
User clicks profile icon/name
  v
Compact account menu opens
  v
Account settings opens Settings
  v
Tabs show Account, Security, Payday, Demo Bank, Data & Privacy, AI Settings and Notifications
```
